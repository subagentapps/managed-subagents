# Webapp Brutalist-Terminal Mobile-First Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current bland landing page at managedsubagents.com with a brutalist-terminal aesthetic that targets developers evaluating "should I install this," renders flawlessly on iPhone 16 Pro Max (430×932 CSS px), and adapts gracefully up through desktop. Includes a self-referential live counter ("this site was built by N subagents").

**Architecture:** Single-route Vite + React 18 SPA (already scaffolded in `webapp/`). Deploys via Workers Static Assets (no SSR). Mobile-first CSS with `min-width` breakpoints — base styles target 430px, then expand at 600/900/1200. Plain CSS modules; no Tailwind, no motion library, no UI kit (per "stay strict on deps"). The "live counter" reads from a static JSON file generated at build time from the dispatch_log of this very site's deploy chain — no runtime API call needed. All animations are CSS-only.

**Tech Stack:** React 18 + TypeScript + Vite + Workers Static Assets. Vitest + jsdom + Testing Library for component tests. Playwright for one mobile-viewport visual smoke test. Build runs `npm run build` → `dist/` → `wrangler deploy`.

## Design tokens (locked here so every task references the same values)

These go into `App.module.css` as CSS custom properties on `:root`:

```css
--bg:        #000000;          /* pure black, no fakery */
--fg:        #d4d4d4;          /* terminal off-white */
--fg-bright: #ffffff;          /* hero text only */
--accent:    #00ff5f;          /* phosphor green, used sparingly */
--warn:      #ffb000;          /* amber for accent secondary */
--rule:      #2a2a2a;          /* hairline borders */
--mono:      ui-monospace, "SF Mono", "Menlo", "Cascadia Mono", monospace;
--gap:       1rem;             /* base vertical rhythm */
--gutter-mobile:  20px;        /* iPhone 16 Pro Max safe-area-conscious */
--gutter-desktop: max(40px, calc((100vw - 1100px) / 2));
```

**Breakpoints (mobile-first):**
- base: 0–599 (iPhone 16 Pro Max = 430 wide; this is the primary canvas)
- `@media (min-width: 600px)`: tablet-portrait expansions
- `@media (min-width: 900px)`: stack columns side-by-side
- `@media (min-width: 1200px)`: max-width container
